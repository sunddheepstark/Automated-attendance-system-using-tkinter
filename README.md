**Attendance Management System**
A GUI-based attendance tracking application built using Python and Tkinter.
**Features**
- Mark students as present or absent
- Automatic absentee marking
- Daily CSV file generation
- Attendance summary and overall percentage calculation
- Real-time logging with timestamps
** Tech Stack**
- Python
- Tkinter
- CSV file handling
- Glob module
**How to Run**
1. Ensure Python is installed.
2. Place `students.csv` in the required directory.
3. Run:python attendance.py
